# Strongman

## Method Bias
- Prioritize event practice over barbell-only substitutes.
- Program implements, carries, loading, and medleys.
- Keep exposure specific to the contest demands.

## Programming Notes
- Use the actual implement when possible.
- When the exact implement is not available, simulate the event as closely as possible using the best available analog.
- Choose the analog by matching the event's implement, loading position, range of motion, movement path, grip demand, event duration, and fatigue profile.
- Train grip, bracing, event transitions, and fatigue resistance.
- Preserve the order and rules of the event when rehearsing.
- Use the broader strongman knowledge base to pick the closest practical substitute, not just the closest barbell movement.

## Contest Prep
- Build the block so it peaks at the contest date.
- Rehearse the exact contest implements as much as possible, especially if the event includes a log press, axle press, yoke, sandbag, or farmers.
- If the exact implement is unavailable, rehearse the closest simulation that preserves the same limiting qualities.
- Rehearse event order and pacing.
- Keep novelty low near the contest.
- Cut back accessory fatigue so event work stays sharp.
