# Form Check Rubric

## Universal Checks
- Stance or setup match the lift.
- Bracing and torso position stay organized.
- Bar path stays efficient and consistent.
- Range of motion is appropriate for the lift and the athlete's goal.
- Timing and control do not collapse under load.
- The limiting factor is the intended one, not a preventable technical leak.

## Lift-Specific Checks
- Squat: depth, knee tracking, torso angle, brace, balance, and bar path.
- Bench: setup symmetry, touch point, elbow path, bar path, and leg drive.
- Deadlift: starting position, wedge, back angle, bar drift, and lockout.
- Overhead or log: clean, rack position, brace, dip and drive, and lockout.
- Weightlifting: start position, pull timing, turnover, catch, and recovery.
- Strongman events: event setup, implement path, speed, grip, and transitions.

## Prioritization
- Fix the biggest limiter first.
- If multiple issues are present, separate them into setup, execution, and fatigue.
- Do not bury the user in tiny corrections if one clear fix would matter more.
