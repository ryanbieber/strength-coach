# Bulgarian Inspired

## Gist
- High-frequency classic lift template inspired by Bulgarian-style daily practice.
- Built around repeated exposures, heavy singles, and technical consistency.

## Best Use
- Advanced lifters with strong recovery and technique.
- Peaking blocks where frequency and specificity matter.

## Phase Bias
- Primary phase: Peak.
- Use it only when the athlete already has enough technical base to tolerate extreme specificity.

## Weekly Shape
- Frequent snatch and clean and jerk exposures
- Squat support work
- Minimal unnecessary variation

## What To Steal
- Frequent competition lift practice.
- Heavy but controlled singles.
- Narrow exercise selection.

## Blend Notes
- Use carefully because recovery demands are high.
- This is a late-stage specificity tool, not a broad developmental block.
