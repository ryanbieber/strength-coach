# Strongman Vertical Pull and Load

## Gist
- Block for events that reward strong pulls, clean pickups, and loading skill.
- Built for athletes who need more force from the floor and less wasted motion.

## Best Use
- 5 day weeks.
- Contests with a pull event and a load or carry event.

## Phase Bias
- Primary phase: SPP.
- Best once the athlete has enough base to handle frequent event practice.

## Weekly Shape
- Day 1: main pull
- Day 2: load event
- Day 3: upper back and trunk
- Day 4: carry or secondary pull
- Day 5: light event rehearsal

## What To Steal
- Contest-specific start position.
- Repeated load practice from the floor or lap.
- Support work that reinforces the same pull angle.

## Blend Notes
- If the exact event object is missing, keep the same loading path and fatigue cost.
- Start with a little more support work, then bias the exact pull and load demands as the contest approaches.
