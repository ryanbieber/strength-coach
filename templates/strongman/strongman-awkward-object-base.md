# Strongman Awkward Object Base

## Gist
- Base template for awkward-object handling before more specific contest work.
- Emphasizes sandbags, stones, kegs, and other objects that force clean setup.

## Best Use
- 4 day weeks.
- Off-season athletes building tolerance to awkward loads.

## Weekly Shape
- Day 1: awkward object carry or load
- Day 2: hinge and squat support
- Day 3: pressing or upper back
- Day 4: repeat awkward-object work

## What To Steal
- Repeated awkward-object exposure.
- Simple base strength around the event slots.
- Enough variety to keep joints happy.

## Blend Notes
- Move toward the contest implement once the event list is fixed.
