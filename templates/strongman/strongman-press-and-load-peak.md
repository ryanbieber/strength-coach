# Strongman Press and Load Peak

## Gist
- Peak block for a press event plus a loading event.
- Built for athletes who need to keep overhead strength and awkward-object work in one cycle.

## Best Use
- 4 day weeks.
- Athletes with a press and a loading event in the same contest.

## Phase Bias
- Primary phase: Peak.
- Use it after an SPP block has already built the relevant press and loading skill.

## Weekly Shape
- Day 1: press event
- Day 2: load event
- Day 3: upper back and triceps
- Day 4: secondary event rehearsal or light carry

## What To Steal
- Event-specific work at the start of the session.
- Small but targeted support work.
- Stable implements close to the show.

## Blend Notes
- Keep the clean, lap, and load positions close to the contest version.
- Trim extra support volume as the show gets near so the event work stays sharp.
