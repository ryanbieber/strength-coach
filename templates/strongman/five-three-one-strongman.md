# 5/3/1 Strongman

## Gist
- Long-term strongman template using 5/3/1 style progression for the main lifts and events.
- Good for athletes who want a sustainable base with event-specific assistance.

## Best Use
- Off-season or early prep.
- Athletes who need long-term consistency.

## Phase Bias
- Primary phase: GPP.
- Use this to build broad strength before switching to more event-specific work.

## Weekly Shape
- Main lift or event on each day
- Supplemental strongman event work
- Limited but targeted accessories

## What To Steal
- Conservative progression.
- Easy-to-run long-term structure.
- Event-specific accessories instead of generic bodybuilding volume.

## Blend Notes
- Swap the supplemental work to match the contest implements.
- The main lifts stay general here, while event work stays present but secondary until an SPP block is needed.
