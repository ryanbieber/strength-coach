# Strongman Two-Event Focus

## Gist
- Narrow block built around two priority events only.
- Useful when the contest list is known and everything else is secondary.

## Best Use
- 3 to 4 day weeks.
- Athletes peaking for a short event list.

## Phase Bias
- Primary phase: Peak.
- Use it when the contest is close and the event list is narrow enough to justify a very specific block.

## Weekly Shape
- Day 1: first priority event
- Day 2: second priority event
- Day 3: support work for both
- Day 4: optional repeat of the weaker event

## What To Steal
- Narrow event selection.
- Higher frequency on the key lifts.
- Minimal extra fatigue.

## Blend Notes
- Cut everything that does not support the two main events.
- This is the late-cycle version of specificity: narrow, direct, and low-noise.
