# Strongman Long Contest Cycle

## Gist
- Longer contest-prep block for athletes with more runway before the show.
- Starts broader, then narrows toward the competition events.

## Best Use
- 5 day weeks.
- Longer prep windows where base work and specificity both matter.

## Weekly Shape
- Day 1: main event
- Day 2: support strength
- Day 3: second event
- Day 4: carry, load, or medley
- Day 5: recovery and light event work

## What To Steal
- Broader work early, narrower work late.
- Stable event slots.
- Enough recovery to keep the block moving.

## Blend Notes
- Shift from support work into contest-specific work as the show approaches.
