# Strongman Program Catalog

This folder holds external strongman programs worth studying when building or reviewing a strongman block.

These are reference points, not fixed scripts. Use them to understand event specificity, fatigue management, and how good strongman plans peak.

## Core Picks

- Joey Szatmary, `12 Week Strongman Program`
  - A 12-week strongman program that can be customized to the athlete's goals or competition.
  - Useful when you want a clear, event-specific peak with explicit singles work.
  - Source: https://szat-strength.myshopify.com/products/strongman-training-template-2019
- Barbell Medicine, `Strongman Template`
  - A 15-week strongman program focused on strength, conditioning, and work capacity.
  - Useful when you want a broader strongman template with event-style work and flexible exercise choices.
  - Source: https://www.barbellmedicine.com/shop/training-templates/strength/strongman-template/

## Also Worth Studying

- Barbell Medicine, `Strength II`
  - A broader strength template that can still support strongman base building.
  - Useful when the athlete needs more general strength before a more event-specific block.
  - Source: https://www.barbellmedicine.com/shop/training-templates/strength/strength-ii-template/
