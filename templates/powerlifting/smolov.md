# Smolov

## Gist
- Aggressive squat-focused peaking template with distinct phases.
- High frequency and high stress.

## Best Use
- A hard squat peak when the athlete can recover from a lot of work.
- Shorter blocks where specificity and overload matter more than long-term sustainability.

## Phase Bias
- Primary phase: Peak.
- Use it only when the athlete is already prepared for a narrow, high-stress squat block.

## What To Steal
- Clear phase structure.
- Intensity ramp into a test.
- Narrow exercise selection.

## Blend Notes
- Use sparingly.
- Better as a peaking concept than a year-round template.
- This is not base building. It is late-stage, narrow, high-cost specialization.
