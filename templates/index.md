# Training Templates

Use these as ingested archetypes, not fixed scripts. Blend them to the athlete's sport, recovery, session length, and target date.

## Archetypes
- [general-strength/index.md](general-strength/index.md)
- [powerlifting/index.md](powerlifting/index.md)
- [strongman/index.md](strongman/index.md)
- [weightlifting/index.md](weightlifting/index.md)
- [bodybuilding/index.md](bodybuilding/index.md)

## Use
- Start at the archetype folder that matches the athlete's goal.
- Use the folder index to pick a template family.
- Blend the chosen template to the athlete's recovery, schedule, and target date.
